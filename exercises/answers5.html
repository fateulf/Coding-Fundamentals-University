Exercise 5

Rewrite the following JavaScript code as a single statement.
var oceans = new Array();
oceans[0] = "Pacific";
oceans[1] = "Atlantic";
oceans[2] = "Indian";
oceans[3] = "Southern";
oceans[4] = "Arctic";

Answer:
var oceans = new Array("Pacific", "Atlantic", "Indian", "Southern", "Arctic");

Exercise 6

What is the value of the length property associated with each of the following arrays?
(a) var books = new Array();
(b) var months = new Array(12);
(c) var musicians = new Array("John", "Paul", "George", "Ringo");
(d) var ages = new Array(21, 20, 25);

Answer:
(a) 0
(b) 12
(c) 4
(d) 3
