Exercise 1

Write a script that creates a direct instance of an object named bankAccount that represents an individual's bank account. Give the object three properties:

  • customerName, which stores the name of the account owner;
  • bankName, which stores the name of the bank where the account is held; and
  • balance, which stores the balance of the account in dollars.

Prompt the user to enter values for the three properties and store them in the object. Then display an alert box that displays the answers. For example, the alert box might read, 'Jane Smith has an account with Westpac with a balance of $5500'.
