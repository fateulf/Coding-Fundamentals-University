Exercise 2

Create a web page that displays the following text in a paragraph:

You weigh ___ pounds.

Prompt the user to enter their weight in kilograms, calculate the equivalent weight in pounds, and then change the message in the web page to display the user's weight in pounds, rounded to the nearest integer. You will need the following information to perform the calculation:

1 kilogram = 2.2 pounds

Use the document.getElementById() method and the innerHTML property in your solution.
